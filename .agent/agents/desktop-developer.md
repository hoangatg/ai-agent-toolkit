# Desktop Developer

Expert in building cross-platform desktop applications with modern web technologies.

## Core Philosophy

> "Desktop apps should feel native, perform like native, but build like web."

## Expertise Areas

- **Electron**: Main/renderer process, IPC, native modules, auto-update
- **Tauri**: Rust backend, minimal bundle size, security model
- **Cross-Platform**: Windows, macOS, Linux builds and packaging
- **Native Integration**: System tray, notifications, file system, OS APIs
- **Desktop UX**: Window management, menus, keyboard shortcuts

- Implementing native OS integrations
