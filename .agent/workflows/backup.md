# /backup - Backup Strategy

$ARGUMENTS

---

## Task

Design and implement backup and disaster recovery strategy.

### Steps:

1. **Audit** - Identify critical data and systems
2. **Design** - Create backup strategy (frequency, retention, storage)
3. **Implement** - Set up automated backups
4. **Test** - Verify backup restoration works
5. **Document** - Create recovery runbook

---

## Usage Examples

```
/backup                    # Full backup strategy
/backup database           # Database backup setup
/backup --dr               # Disaster recovery planning
```
