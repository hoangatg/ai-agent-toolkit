# /prompt - Prompt Engineering

$ARGUMENTS

---

## Task

Design and optimize prompts for AI features.

### Steps:

1. **Define** - Clarify the AI task and desired output
2. **Design** - Craft initial prompts using best practices
3. **Test** - Run prompts against diverse inputs
4. **Evaluate** - Score output quality and consistency
5. **Optimize** - Iterate on prompt structure and examples

---

## Usage Examples

```
/prompt classification     # Design classification prompts
/prompt extraction         # Design data extraction prompts
/prompt --eval             # Evaluate existing prompts
```
