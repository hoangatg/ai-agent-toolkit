# /docs - Documentation Generation

$ARGUMENTS

---

## Task

Generate comprehensive project documentation.

### Steps:

1. **Analyze** - Scan codebase for functions, components, APIs
2. **Generate** - Create documentation using `technical-writer` agent
3. **Organize** - Structure docs by category (API, guides, reference)
4. **Review** - Verify accuracy and completeness

---

## Usage Examples

```
/docs                      # Generate full project docs
/docs api                  # Generate API documentation
/docs components           # Document React components
```
