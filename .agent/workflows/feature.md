# /feature - Feature Development

$ARGUMENTS

---

## Task

Implement a complete feature end-to-end.

### Steps:

1. **Plan** - Define scope, acceptance criteria, task breakdown
2. **Implement** - Build DB → API → UI using appropriate agents
3. **Test** - Write and run tests for all layers
4. **Review** - Self-review using code-reviewer agent
5. **PR** - Create clean pull request with description

---

## Usage Examples

```
/feature user-profile      # Build user profile feature
/feature search --fullstack # Full-stack search feature
```
