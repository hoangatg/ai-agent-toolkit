# /ai - AI Integration

$ARGUMENTS

---

## Task

Add AI-powered features to the application.

### Steps:

1. **Define** - Determine AI feature type (chat, search, generation)
2. **Design** - Use `ai-ml-engineer` agent for architecture
3. **Implement** - Build AI pipeline (prompts, embeddings, retrieval)
4. **Test** - Create evaluation suite for AI output quality
5. **Optimize** - Tune prompts, caching, and costs

---

## Usage Examples

```
/ai chatbot                # Add AI chatbot feature
/ai rag                    # Build RAG pipeline
/ai embeddings             # Set up semantic search
```
