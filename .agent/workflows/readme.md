# /readme - README Generator

$ARGUMENTS

---

## Task

Generate a professional, comprehensive README.

### Steps:

1. **Analyze** - Scan project structure, package.json, configs
2. **Generate** - Create README with sections: Overview, Install, Usage, API, Contributing
3. **Polish** - Add badges, screenshots, table of contents
4. **Review** - Verify accuracy against actual project

---

## Usage Examples

```
/readme                    # Generate full README
/readme --update           # Update existing README
/readme --badges           # Add CI/coverage badges
```
