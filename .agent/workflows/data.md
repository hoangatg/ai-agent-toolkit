# /data - Data Pipeline

$ARGUMENTS

---

## Task

Design and implement a data pipeline.

### Steps:

1. **Source** - Identify data sources and formats
2. **Design** - Use `data-engineer` agent for pipeline architecture
3. **Build** - Implement extraction, transformation, loading
4. **Quality** - Add validation and monitoring
5. **Schedule** - Set up automation and orchestration
