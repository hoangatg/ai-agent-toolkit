# /i18n - Internationalization

$ARGUMENTS

---

## Task

Add or improve internationalization support.

### Steps:

1. **Audit** - Detect hardcoded strings in codebase
2. **Setup** - Configure i18n library (next-intl, react-i18next, etc.)
3. **Extract** - Move strings to locale files
4. **Implement** - Add language switcher and locale routing
5. **Verify** - Test all supported languages

---

## Usage Examples

```
/i18n setup               # Initial i18n setup
/i18n audit                # Find hardcoded strings
/i18n add-locale vi        # Add Vietnamese locale
```
