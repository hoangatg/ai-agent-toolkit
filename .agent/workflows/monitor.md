# /monitor - Setup Monitoring

$ARGUMENTS

---

## Task

Set up monitoring, alerting, and observability.

### Steps:

1. **Define** - Identify key metrics and SLOs
2. **Instrument** - Add logging, metrics, and tracing
3. **Dashboard** - Create monitoring dashboards
4. **Alert** - Configure alerts for critical thresholds
5. **Test** - Verify alerts fire correctly

---

## Usage Examples

```
/monitor                   # Setup monitoring stack
/monitor --slo             # Define SLOs and error budgets
/monitor --alerts          # Configure alert rules
```
