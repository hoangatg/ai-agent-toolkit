# /optimize - Code Optimization

$ARGUMENTS

---

## Task

Optimize application performance and resource usage.

### Steps:

1. **Measure** - Bundle analysis, Lighthouse, profiling
2. **Identify** - Find top optimization opportunities
3. **Apply** - Implement optimizations (tree-shaking, lazy loading, caching)
4. **Verify** - Re-measure to confirm improvements

---

## Usage Examples

```
/optimize                  # Full optimization audit
/optimize --bundle         # Bundle size optimization
/optimize --images         # Image optimization
```
